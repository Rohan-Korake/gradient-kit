<h1 align="center">GradientKit</h1>

GradientKit is a modern gradient generator built with **React**, **Vite**, and **Tailwind CSS**. Create, customize, preview, copy, save, and export beautiful CSS & TailwindCSS gradients with a clean and intuitive interface.

---

## Features

- Create Linear, Radial, and Conic gradients
- Unlimited color stop customization
- Live gradient preview
- Adjustable gradient angle
- One-click copy for CSS & Tailwind CSS code
- Save your favorite gradients
- Export gradients
- Fully responsive design
- Fast performance with Vite
- Clean and modern UI

---

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- Zustand (State Management)

---

## Getting Started

```bash
git clone https://github.com/Rohan-Korake/gradient-kit.git
```

---

### Navigate to the project

```bash
cd gradient-kit
```

---

### Install dependencies

```bash
npm install
```

---

### Start the development server

```bash
npm run dev
```

---

## Build for Production

```bash
npm run build
```

---

## Preview the production build:

```bash
npm run preview
```
